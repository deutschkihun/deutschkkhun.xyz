## 1. Data Structure

![ds2](/static/ds-basic/ds1.png)

- 정수 integer / 실수 real number

선형구조

한 원소 뒤에 하나의 원소 만이 존재하는 형태로 자료들이 선형으로 나열되어 있는 구조를 가진다.

![ds2](/static/ds-basic/ds2.png)

비선형구조
원소 간 다대다 관계를 가지는 구조로 계층적 구조를 표현하기에 적절하다.

![ds3](/static/ds-basic/ds3.png)

선형구조와 비선형구조의 가장큰 차이점

- 자료를 구성하는 원소들은 순차적으로 나열시킨 형태를 의미한다 (The elements that make up the material are the sequentially arranged forms of)

- 하나의 자료 뒤에 여러개의 자료가 존재할 수 있는 형태를 의미한다.(This refers to a form in which multiple resources can exist after a single resource.)
